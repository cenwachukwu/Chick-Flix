# Chick-Flix

## Description

A purely javaascript based learning application which features flashcard decks and a scoring quiz game.

## Technologies:
* HTML
* JavaScript
* CSS
* VS code

## Required Installations:
* Fork and clone (or just clone) the repository
    * `git clone https://github.com/cenwachukwu/Chick-Flix`
    * cd into the directory
* To open the code in vs code, run code .
